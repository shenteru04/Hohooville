# Quick Reference - Archive & Inactive User Management

## At a Glance

| Action | Location | Button | Result |
|--------|----------|--------|--------|
| **Archive User** | User Management | Red "Archive" | Removes from management, goes to System Settings |
| **Unarchive User** | System Settings > Archived | Green "Unarchive" | Restores to User Management |
| **Make Inactive** | User Management Edit | Status: "Inactive" | Stays in User Management, shows as inactive |
| **Reactivate User** | User Management Edit | Status: "Active" | Changes from inactive to active |
| **View All Archived** | System Settings > User Archival | N/A | Shows archived trainers and trainees |
| **View Inactive Users** | User Management > Inactive Tab | N/A | Shows users marked as inactive |

---

## Step-by-Step: Archive a User

### When to Archive
Use this when a trainer or trainee leaves permanently or course is complete and you want to remove them from active management.

### How to Archive
1. **Go to User Management** page
2. **Find the user** you want to archive (in Admin/Trainer/Trainee/Registrar tab)
3. **Click the red "Archive" button** next to their name
4. **Confirm** the action in the popup
5. ✅ **Done** - User is now archived

### Result
- User disappears from User Management
- User appears in System Settings > User Archival > Archived Trainers/Trainees
- You cannot edit the user from User Management anymore
- User data is safe (not deleted)

---

## Step-by-Step: Unarchive a User

### Why Unarchive
If you archived a user by mistake or need to restore their account.

### How to Unarchive
1. **Go to System Settings**
2. **Click "User Archival"** section
3. **Find the "Archived Trainers"** or **"Archived Trainees"** tab
4. **Locate the user** you want to restore
5. **Click the green "Unarchive" button**
6. **Confirm** the action
7. ✅ **Done** - User is restored to User Management

### Result
- User reappears in User Management with their original status
- User is ready to be edited again
- All their original data is intact

---

## Step-by-Step: Set User to Inactive

### When to Set Inactive
Use this when you want to temporarily disable a user (e.g., on leave, suspension) but might reactivate later.

### How to Set Inactive
1. **Go to User Management** page
2. **Find the user** (in their role tab)
3. **Click the blue "Edit" button**
4. **In the Edit Modal**, find the **Status** dropdown
5. **Change from "Active" to "Inactive"**
6. **Click "Save User"** button
7. ✅ **Done** - User is now inactive

### Result
- User stays in User Management (doesn't disappear)
- User appears in the **"Inactive Users"** tab
- User shows an "Inactive" badge
- User can still be edited
- User can be quickly reactivated

---

## Step-by-Step: Reactivate an Inactive User

### Why Reactivate
When the inactive user returns or no longer needs suspension.

### How to Reactivate
1. **Go to User Management**
2. **Click the "Inactive Users" tab**
3. **Find the user** you want to reactivate
4. **Click the blue "Edit" button**
5. **In the Edit Modal**, find the **Status** dropdown
6. **Change from "Inactive" to "Active"**
7. **Click "Save User"** button
8. ✅ **Done** - User is now active

### Result
- User moves from "Inactive Users" tab to the appropriate role tab (Admin/Trainer/etc.)
- "Inactive" badge disappears
- User can log in again
- User is fully active

## Understanding the Difference

### ARCHIVE (Permanent Removal)
```
✓ User goes to System Settings
✓ Only recoverable from System Settings
✗ Not editable in User Management
✓ Best for: Course complete, trainer left, no longer needed
```

### INACTIVE (Temporary Disable)
```
✓ User stays in User Management
✓ Easy to reactivate
✓ Still editable
✓ Best for: On leave, suspension, temporary access denial
```

## Finding Users

### Find in User Management
1. Go to **User Management**
2. Choose the role tab (Admin/Trainer/Trainee/Registrar)
3. Use **Search box** to find by username or email
4. Use **Status filter** to show Active or Inactive only
5. Use **Pagination** to browse pages

### Find in System Settings
1. Go to **System Settings**
2. Go to **User Archival** section
3. Choose tab:
   - **Inactive Users** - temporarily disabled users who are editable
   - **Archived Trainers** - permanently archived trainer accounts
   - **Archived Trainees** - permanently archived trainee accounts

### Search Tips
- **Search is case-insensitive** (Smith = smith)
- **Works on username or email** (test123 or test@example.com)
- **Searches only current tab** (admin names won't show in trainer tab)
- **Status filter is per-tab** (each tab remembers its filter)

**Steps**:
1. Go to User Management > Trainer tab
2. Find trainer name

**Goal**: Review who's inactive

**Steps**:
1. Go to System Settings
2. Go to **User Archival** section

## Frequently Asked Questions

### Q1: I Archived a User by Mistake, Can I Get Them Back?
**A**: Yes! Go to System Settings > User Archival > [Archived Trainers/Trainees] > Find user > Click "Unarchive"

### Q2: What's the Difference Between Inactive and Archive?
**A**: 
- **Inactive**: Temporary, user stays in User Management, can still edit, easy to reactivate
- **Archive**: Permanent, user removed from User Management, only recoverable from System Settings

### Q3: Can I Archive an Inactive User?
**A**: The Archive button is only visible for Active users. Inactive users need to be reactivated first if you want to archive them. Or you can reactivate them from System Settings if they're already archived.

### Q4: Do Archived Users Get Deleted?
**A**: No! All data is preserved. Archived users are just hidden from normal User Management. You can restore them anytime from System Settings.

### Q5: What Happens to a User's Data When Archived?
**A**: Nothing is deleted. Their profile data, grades, enrollments, and all related records remain unchanged. Only their visibility in User Management changes.

### Q6: Can I Search Archived Users in User Management?
**A**: No, archived users don't appear in User Management at all. To find archived users, go to System Settings > User Archival section.

### Q7: Can I Unarchive from User Management?
**A**: No, unarchive is only available in System Settings > User Archival section.

### Q8: What Tabs Are in User Archival?
**A**: 
1. **Inactive Users** - Temporarily disabled users (still editable)
2. **Archived Trainers** - Permanently archived trainer accounts
3. **Archived Trainees** - Permanently archived trainee accounts

### Q9: Can I Archive Myself (My Own Account)?
**A**: Technically yes, but you won't be able to unarchive yourself afterwards since you won't have access. Don't do this!

### Q10: Is There An Undo Button After Archiving?
**A**: No, but you can unarchive from System Settings. Just go there and click "Unarchive".

## Troubleshooting

### Problem: Archive Button Not Visible
**Solution**: 
- Make sure user status is "Active" (not already inactive)
- Make sure you're logged in as Admin
- Try refreshing the page

### Problem: User Still Shows After Clicking Archive
**Solution**:
- Wait a few seconds for page refresh
- Try clicking browser refresh button
- Check if it's in a different role tab than expected

### Problem: Can't Find Archived User
**Solution**:
- Go to System Settings > User Archival
- Make sure you're looking in correct tab (Trainers vs Trainees)
- Use search if available
- Check System Settings is loading properly

### Problem: Edit Button Doesn't Work for Inactive User
**Solution**:
- Edit button should work for inactive users in User Management
- If it doesn't, try page refresh
- Make sure you're clicking the "Edit" button (not row itself)
